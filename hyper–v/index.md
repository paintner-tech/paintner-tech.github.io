

# Einleitung
Hyper-V ist die Virtualisierungslösung von Microsoft und Bestandteil von Windows Server sowie Windows Pro/Enterprise Editionen.

Mit Hyper-V lassen sich virtuelle Maschinen (VMs) erstellen und betreiben, um verschiedene Betriebssysteme parallel auf einem Host auszuführen.

Ich beschäftige mich mit Hyper-V, da neben Linux-, Proxmox- und VMware-Umgebungen auch Microsoft-Virtualisierung im industriellen Umfeld und bei Kunden zunehmend gefragt ist.
